Detailed Description

Module gérant les boxes de collisions, ainsi que leurs fonction associées.

Une BoxColliders contient différente type de boxes, associés a chaque images delimitant les zones ou l'on peut subir des coups, ou en infliger, ainsi que les collisions avec l'environnement.
